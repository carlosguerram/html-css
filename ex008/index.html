começarei a dissertar sobre formatação

começando com NEGRITO

neste parágrafo usarei a tag <b> que não possui semântica, mas adiciona o negrito:
frase em negrito

já para adicionar destaque, usa-se a tag <STRONG> como pode ver no exemplo:
texto em negrito
texto em negrito*
onde o asterisco representa a palavra com o uso da tag <b>

da mesma forma, podemos usar a tag <i> para criar palavras em itálico !

mas é recomendável utilizar a tag <EM> pois é mais recente e tambem deixa em itálico.

ainda sobre marcação. podemos usar a tag <mark> para criar uma área marcada

podemos também usar a tag <big>para criar palavras maiores

mas essa tag se encontra obsoleta pois pode ser efetuada também pelo CSS

a tag <small> já, serve para modificar palavras para pequenas

textos deletados podem ser colocados com a tag <del>

assim como há duas formas de adicionar um texto sublinhado. uma semãntica usando a tag <ins>


e outra não semãntica usando a tag <u>
